# 🎮 2xKO Fan Forum & Info Hub

Un foro y portal informativo hecho por fans dedicado al videojuego **2xKO**.

---

## 📌 Descripción del proyecto

Este proyecto es un **foro web + blog informativo** centrado en *2xKO*. Su objetivo es crear un espacio donde jugadores puedan:

* Consultar guías y builds
* Mantenerse al día con noticias y actualizaciones

Incluye funcionalidades como:

* Sistema de registro e inicio de sesión
* Categorías (noticias, guías, competitivo, etc.)
* Panel de administración básico

El proyecto nace como una iniciativa académica y práctica para aprender desarrollo web moderno, trabajo en equipo y despliegue de aplicaciones reales.
